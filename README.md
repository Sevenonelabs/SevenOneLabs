<details>
<summary><b>SquarePic</b> — <i>Privacy-first image editing platform</i></summary>
<br />

SquarePic is a lightweight, open-source image editor designed for social media platforms. Built with privacy at its core — zero uploads, zero tracking, all processing happens client-side.

| Attribute | Details |
| :--- | :--- |
| **Stack** | TypeScript, Next.js, Tailwind CSS, Canvas API |
| **Scale** | Client-side processing, single-page application |
| **Performance** | Instant processing, no server round-trips |
| **Security** | Zero data upload, fully client-side, no storage |
| **Impact** | Privacy-first alternative to cloud-based editors |
| **Repository** | [github.com/Sevenonelabs/square_pic](https://github.com/Sevenonelabs/square_pic) |

SquarePic redefines how users interact with image editing by eliminating the privacy concerns inherent in traditional cloud-based tools. Every crop, resize, and filter operation executes locally in the browser, ensuring user data never leaves the device.

</details>
